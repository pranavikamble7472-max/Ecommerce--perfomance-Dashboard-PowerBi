# Ecommerce Performance Dashboard - Power BI

## 📊 Project Overview
Interactive Power BI dashboard built to analyze E-Commerce business performance. 
This dashboard analyzes 2M+ revenue data to identify sales trends, top products, and regional performance.

## 📈 Key Metrics
- **Total Revenue**: 2M
- **Total Orders**: 500
- **Average Order Value**: 3.53K

## 📉 Key Insights
- **Sales Trend**: Monthly revenue and order trend analysis
- **Category Performance**: Top performing product categories
- **Regional Analysis**: Region-wise sales and revenue breakdown
- **Top Products**: Best selling products identification

## 🛠 Tools & Skills Used
- Power BI Desktop
- DAX Measures
- Data Cleaning & Modeling
- Data Visualization
